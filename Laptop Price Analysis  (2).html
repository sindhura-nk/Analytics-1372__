Steps to follow for Data Analysis Projects¶

  1. Understand the Data - Read the descriptions of the columns, understand what kind of data is being stored
  2. Data Gathering
  3. Check the data quality - Duplicated rows, Missing content, Outliers
  4. Data Cleaning - handling of missing data - pipelines Duplicated row handling, Missing data handling,Handling outliers
  5. Statistical Analysis - describe() - categotrical data as well as for continuous data
  6. Descriptive Analysis Univariate Analysis - Analysis Bivariate Analysis Multivariate Analysis
  7. Documentation - Include a cell where you discuss all the Insights gathered. You can mention Recommendations also.

# Import Pipeline
from sklearn.pipeline import make_pipeline
# Import Imputer that performs missing data handling by replacing missing data with central tendency
from sklearn.impute import SimpleImputer
# Column Transformer combines pipelines. Here we have 2 pipelines- categorical and continuous 
from sklearn.compose import ColumnTransformer
In [17]:
# pipeline to handle numerical missing content
num_pipe = make_pipeline(SimpleImputer(strategy="median"))

# pipeline to handle categorical missing content
cat_pipe = make_pipeline(SimpleImputer(strategy="most_frequent"))
In [18]:
# combine both the pipelines
final_pipe = ColumnTransformer([
    ('cat',cat_pipe,cat),
    ('con',num_pipe,con)
]).set_output(transform="pandas")
