Identity matrix: square matrix

Identity Matrices : The identity array is a square array with ones on the main diagonal.

It has the following properties:

Diagonal elements are all 1s.
Non-diagonal elements are all 0s.

This is mostly used in linear algebra and numerical computations

Random number generation

You can use numpy.random module to create random numbers (float/integer)

np.random.rand(shape) => create random float data based on shape(number of rows and number of columns).If shape is not given,it will create one data point
np.random.randint(range,shape) => creates integer data of given range and shape
np.random.randn(shape) => creates normally distributed data for given shape.If shape is not given,it will create one data point
